The data set is a 3x3 grid which is layout as follow:

-------+--------+-------
X(0,0) | O(1,0) | X(2,0)
-------+--------+-------
O(0,1) | O(1,1) | X(2,1)
-------+--------+-------
X(0,2) | X(1,2) | O(2,2)
-------+--------+-------

The text values are either X or O.

The JSON values are like { x: 0, y: 0, value: 'X'}

The Image are real image with a X and a O of 75x75 pixels.
